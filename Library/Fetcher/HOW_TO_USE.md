# How to Use Fetcher

## Quick Start

1. Open terminal in AutoMates folder
2. Give Fetcher the identity: `read Library/Fetcher/Fetcher_Identity.md`
3. Ask for what you need:

```
Fetch information about React hooks
```

Fetcher finds sources, extracts them, and organizes into `Library/Sources/`.

---

## Examples

```
Fetch OAuth2 security best practices
```

```
Fetch https://docs.python.org/3/tutorial/ to Python
```

```
Prepare a study file for Builder about authentication
```
