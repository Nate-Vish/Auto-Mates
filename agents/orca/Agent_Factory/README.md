# Agent Factory

Templates and specifications for creating new agents.

## Contents

- `Agent_Template.md` - Base template for new agent identities
- Use this folder to design new agents before deploying them

## Creating a New Agent

1. Copy `Agent_Template.md`
2. Fill in the agent's role, workflow, and memory structure
3. Create the agent folder in `agents/`
4. Create Memory_Logs structure
5. Test with `/summon [agent]`
