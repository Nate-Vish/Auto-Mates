# Dubi Memory Logs

Navigation guide for Dubi's persistent memory.

| File | Purpose | Read When |
|------|---------|-----------|
| `Context.md` | Quick startup snapshot | Every session start (read first) |
| `Checkpoint.md` | Save/resume complex tasks | Every session start |
| `Lessons.md` | Mistakes to avoid, patterns that worked | Every session start |
| `Preferences.md` | How the user likes things done | Every session start |
| `Sessions/` | Session history (one file per date) | When needing detailed context |
| `Notes/` | Technical knowledge files | When researching specific topics |
