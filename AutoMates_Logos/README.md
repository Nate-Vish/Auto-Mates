# AutoMates Folder Logos

Custom folder icons for the AutoMates project structure.

---

## How to Apply (macOS)

1. Open the PNG file you want to use
2. **Cmd+A** to select all, **Cmd+C** to copy
3. Right-click the target folder â†’ **Get Info** (or Cmd+I)
4. Click the folder icon in the top-left of the Info window
5. **Cmd+V** to paste
