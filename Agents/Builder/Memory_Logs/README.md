# Memory Logs - Navigation Guide

This folder contains my persistent memory across sessions.

## What Each File Is For

| Location | Purpose | When to Read | When to Write |
|----------|---------|--------------|---------------|
| `Sessions/` | Logs of past conversations | Start of session | End of session |
| `Notes/` | Technical knowledge files | When relevant to task | When learning something |
| `Lessons.md` | Mistakes to avoid, patterns that work | Start of session | When I learn something important |
| `Preferences.md` | How the user likes things done | Start of session | When user expresses preference |
| `Checkpoint.md` | Save progress on complex tasks | Start of session | During multi-step tasks |

## Session Start Protocol

1. Read this README.md first
2. Check Checkpoint.md - any tasks to resume?
3. Read ALL files in Sessions/ folder
4. Read ALL files in Notes/ folder
5. Read Lessons.md
6. Read Preferences.md
7. Then proceed with the current task

## Session End Protocol

1. Update Sessions/ with what happened
2. Update Notes/ if I learned something new
3. Update Lessons.md if I made a mistake or found a pattern
4. Update Preferences.md if user expressed a preference
5. Update Checkpoint.md if task is incomplete
