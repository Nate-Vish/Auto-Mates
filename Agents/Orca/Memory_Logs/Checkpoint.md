# Checkpoint - Orca
**Purpose:** Save and resume complex multi-step tasks.

---

## How to Use This File

### When to Checkpoint
- Before starting a complex multi-step task
- After completing each major step
- When ending a session mid-task
- Before any risky or irreversible operation

### When to Clear
- When the task is fully complete
- When abandoning a task (note why in Memory_Logs)
